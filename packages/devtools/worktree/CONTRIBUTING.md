## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/worktree-wizard.git`
3. Install dependencies: `npm install`
4. Create a branch: `git checkout -b feature/your-feature-name`

## Development Workflow

```bash
# Run in development mode
npm run dev -- list

# Run tests
npm test

# Run linter
npm run lint

# Format code
npm run format

# Build
npm run build
```

## Code Style

- Use TypeScript strict mode
- Follow existing code patterns
- Write meaningful commit messages
- Add tests for new features
- Update documentation when needed

## Pull Request Process

1. Update tests if needed
2. Update README.md if adding features
3. Ensure all tests pass
4. Ensure code is linted and formatted
5. Create a Pull Request with a clear description

## Feature Ideas

We'd love contributions in these areas:

- Additional commands (e.g., `wt prune`, `wt move`)
- More IDE integrations (IntelliJ, Cursor, etc.)
- Git hooks support
- Configuration file support
- Better error handling and messages
- Performance improvements
- Documentation improvements

## Reporting Bugs

Open an issue with:
- Clear description
- Steps to reproduce
- Expected vs actual behavior
- System info (OS, Node version, Git version)

## Code of Conduct

Be respectful, collaborative, and constructive. We're all here to make great software together!
